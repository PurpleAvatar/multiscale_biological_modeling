Setting up CellBlender

Setting up CellBlender can be done using the default tutorial with just a few changes from the original installation instructions.

Follow the instructions from the CellBlender website: https://mcell.org/downloads/windows/install_2019_05/index.html

With the following two changes:

  1. Instead of downloading the newest version of Blender, go to the previous versions tab...

...and download Blender 2.79b

  1. Once Blender is downloaded,  the file path may be different from what is shown in the CellBlender tutorial- instead of  Blender/2.79/python/bin, the pathway may be something like: blender-2.79/2.79/python/bin. Changing the name of the downloaded Blender folder to match the path in the tutorial may help reduce confusion.

Setting-up Simulations

See “CellBlender Layout” for instructions on any underlined items.

Step 0:

  1. From a new Blender file, initialize CellBlender
  2. Delete the existing default cube by right clicking on the cube to select the cube (an orange outline should be around the cube when it is selected) and pressing the “x” key to delete
  3. From the tab CellBlender > Model Objects, insert a new plane (follow figure S.1) 

Figure S.1 - From CellBlender > Model Objects, click on the “(+)” symbol to center the cursor. Next press the square “plane” button to create the object. To have CellBlender recognize this object as a model object, press the “+” button. The name of this object is “Plane” by default, though you can change this and edit the color by selecting the color wheel. A slightly transparent coloring will help with visibility, but is not necessary.

Figure S.2 - Resizing the render preview window (where the objects are visible in the center of the screen) is recommended. From the View menu, select “Top” to align the view directly overhead. With the plane object selected, follow the arrow over to the object parameters menu (the orange cube) and scale the plane by setting the first two values to “1.5”. Then, hover the mouse over the object and either use the ctrl + “+” 6 times or the scroll wheel on the mouse to zoom in.

Rules for Non-NAR (Non-Negative Auto-Regulation)

  1. Go to CellBlender > Molecules and create the following molecules:
  1. Select CellBlender > Molecules
  2. Create a molecule by clicking the “+” button
  3. Optional: change the color
  4. Rename the molecule “Y”
  5. Change molecule type to “Surface Molecule”
  6. Change the diffusion constant to “1e-6”
  7. Change the scale factor to “5.0”

  1. Create another  molecule named “Hidden”
  2. Set the molecule type to “Surface Molecule”  and the diffusion constant to “1e-6”

  1. Go to CellBlender > Molecule Placement to set the following options;

  1. Select CellBlender > Molecule Placement
  2. Add a new release site by pressing the “+” button
  3. Select the Molecule “Hidden”
  4. Type in the name of the Model Object “Plane” (or custom name set in fig. S.1
  5. Set the Quantity to Release to 300

  1. Go to CellBlender > Reactions to create the following reactions:
  1. Select CellBlender > Reactions
  2. Add a new reaction by pressing the “+” button
  3. Under Reactants, type “ Hidden’ “. Note: the apostrophe is required to indicate the directionality of a surface molecule. 
  4. Under Products, type “ Hidden’ + Y’ “.
  5. Set the Forward Rate to “2e2”

  1. Add a new reaction by pressing the “+” button
  2. Under Reactants, type “ Y’ “.
  3. Under Products, type “ NULL“.
  4. Set the Forward Rate to “3e2”

  1. Go to CellBlender > Plot Output Settings to set up a plot as follows:

  1. Select CellBlender > Plot Output Settings
  2. Add a new plot by pressing the “+” button
  3. Under molecule, select  “Y”
  4. If not already selected, choose “Java Plotter” for the plot method.
  5. Optional: select “Molecule Colors” to match the same colors in the plot as the molecules in the animation

  1. Go to CellBlender > Run Simulation and select the following options: un Simulation using 12000 frames, which should take less than 5 seconds to finish
  1. Select CellBlender > Run Simulation
  2. Change the number of iterations to “12,000”
  3. Click “Export & Run”

  1. Press the CellBlender > Reload Visualization Data button

Fig. NN.1 - After pressing the Reload Visualization Data button, the animated simulation can be played using the buttons at the bottom of the screen. The timeline represents each frame of animation, which should now show 12,000 frames.

  1. Go back to CellBlender > Plot Output Settings and click the plot button. NOTE: if no plot displays, see [additional instructions on trying other plotters or the jupyter notebook]

 

Fig. NN.2 - The plot above should be displayed when completing the No-NAR tutorial

Rules for Feedforward Loop

See “Setting-up Simulations” before starting these steps

Quick Tutorial:

  1. Load the file from Non-NAR tutorial
  2. Change the Y molecule to have a diffusion constant of 2e-6
  3. Add the following molecules
  1. Change Y to a diffusion constant of 2e-6
  2. X, diffusion constant of 2e-6
  3. Z, diffusion constant of 2e-6
  1. Add the following reactions
  1. Hidden' -> Hidden' + X', rate: 4e2
  2. X' -> X' + Y', rate: 5e2
  3. X' -> X' + Z', rate: 1e3
  4. Y' + Z' -> Y', rate: 3e4
  5. X' -> NULL, rate: 1e3
  6. Y' -> NULL, rate: 3e2
  7. Z' -> NULL, rate: 5e2
  8. Hidden’ -> NULL, 3e3
